# Anonymous Wifi Chat
## peer-to-peer

A lightweight *Peer-to-Peer* chat application designed for secure and instant communication within a local WiFi network. Built entirely in Go, this application leverages the TCP protocol for reliable message delivery and UDP broadcasting for seamless, automatic room discovery.

### privacy features
The system is architected so that only the Host can monitor peer connections (joins/leaves), while participants remain anonymous to one another, ensuring a focused and private conversational environment.
